Descrizione dettagliata

Classe principale d'avvio del programma per la gestione delle spedizioni. Nel programma è presente una sola finestra (o JFrame) rappresentata dalla MainWindow che si occupa si sostituire dinamicamente le varie schermate (o CustomPanel) presenti nel programma.
